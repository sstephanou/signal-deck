### Weather
Uses [Open-Meteo](https://open-meteo.com/). API documentation can be found [here](https://open-meteo.com/en/docs).
Limits:
* Minutely Limit: 600 calls/min
* Hourly Limit: 5,000 calls/hour
* Daily Limit: 10,000 calls/day
* Monthly Limit: 300.000 calls/month

Keep in mind, even with the current time, OpenMeteo returns the aggrigate weather data of the last 15 minutes.
